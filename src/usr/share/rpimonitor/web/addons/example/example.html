Example addon

Example addon shows how you can develop your own addon for RPi-Monitor.

An addon (named for this example example) is composed by a set of 3 files stored in the subdirectory of the same name
/usr/share/rpimonitor/web/addons/example/:
  • a html page: example.html
  • a css style sheet: example.css
  • a javascript script : example.js
These 3 files are directly used by RPi-Monitor to render a page. Javascript of the addon have a direct access to RPi-Monitor data using the functions in rpimonitor.utils.js and the functions getData('json_data').

This example is showing how to get uptime information and display it using the function Uptime( value ):
